# Webhook Processing Pipeline

## 📌 Overview

A webhook-driven system that processes incoming events asynchronously and delivers results to subscribers.

## 🚀 Tech Stack

* TypeScript
* Express
* PostgreSQL
* pg-boss
* Docker

## ▶️ Run Locally

```bash
npm install
npm run dev
```
